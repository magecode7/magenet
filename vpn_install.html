Руководство по установке VPN

На главную

1. Получите подписку VPN

Для оформления подписки, свяжитесь с нами в Telegram или получите конфигурацию у других провайдеров

2. Установите клиент VPN

Для подключения к VPN, необходимо установить клиент поддерживающий VPN подписку:

  • Hiddify
  • NekoBox
  • v2rayNG
  • Happ

Мы рекомендуем Hiddify, для его установки выполните следующие шаги:

  1. Скачайте и установите клиент Hiddify, подходящий для вашей операционной системы:
    • GitHub
    • Google Play
    • App Store
    • Microsoft Store
  2. Скопируйте полученную конфигурацию подписки
  3. Откройте приложение и выполните следующие действия:
    • Выберите "Новый профиль"
    • Нажмите "Добавить из буфера обмена"
  4. Теперь вы можете подключаться к VPN

Дополнительно

  • По умолчанию Hiddify не проксирует региональные домены и IP-адреса. Чтобы проксировать всё, необходимо в настройках выбрать регион "Другой".
  • На ПК версиях по умолчанию стоит режим "системного прокси". Для работы некоторых программ включите режим "VPN".
  • В разделе "Прокси" при включенном VPN можно выбрать текущую конфигурацию прокси. По умолчанию выбирается та, что имеет наименьшую задержку.
